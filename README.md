# 出荷リスト ジェネレーター（horitori-list）

植木EC の当日（翌日）出荷分の注文データから、掘り作業に必要な **樹木名** と **品番** だけを抽出し、掘り子に渡す作業指示書を **印刷** できる単一 HTML ツールです。インストール不要・ネット不要・サーバー不要。ブラウザでファイルを開くだけで動きます。

## ファイル

- `horitori_v2.html` … **v2 本体（推奨）**。売上台帳の Excel を直接読み込み、出荷日を入れるだけでリストを出力
- `horitori_jidouseisei.html` … v1 本体（Excel から表をコピペして変換する方式）
- `出荷リストツール_使い方.txt` … スタッフ周知用の手順書（v1 の開き方・使い方）

## v2 の使い方（台帳から直接読み込み）

1. `horitori_v2.html` をブラウザ（Edge/Chrome）で開く
2. 出荷日を確認（初期値は **翌日**）
3. 「台帳から読み込む」→ 初回だけ OneDrive 同期済みの `売上台帳.xlsx` を選ぶ（次回からは記憶。環境によっては毎回選択ダイアログが出る）
4. 一覧が出る → 「印刷する」→ 掘り子へ渡す

### v2 の仕組み・仕様

- 台帳のシートは自動判定：見出しに **出荷日・出荷元・商品名** を持つ、データ最多のシート（現状 `顧客購入履歴_20231221～`）
- 抽出条件：**出荷日＝入力日 かつ 出荷元＝「自社」**（出荷元は「設定」で変更・保存可能）
- 品番は商品名から正規表現で抽出（取れない場合のみ品番列で補完）。**数量** 列を表示し、2 以上は強調（画面：色付き／印刷：太字＋下線）
- 台帳ファイルの **最終更新日時** を表示し、6 時間以上古い場合は警告（OneDrive 同期遅れによる古いリストの印刷防止）
- Excel の読み取り（SheetJS）はファイル内に同梱。**ネット接続・サーバー・インストール不要** は v1 と同じ
- 台帳が読めないとき用に、v1 と同じ **貼り付け変換** を「予備手段」として搭載

## 使い方

1. `horitori_jidouseisei.html` をダブルクリックして開く
2. 出荷日を確認（初期値は **翌日**）
3. 注文データ（出荷リスト）の表をコピーして貼り付ける
4. 「変換する」→ 樹木名・品番の一覧が出る
5. 「印刷する」→ 掘り子へ渡す

## 抽出ルール（商品名フリーテキストから）

- **品番**: `英字+数字-数字`（例 `dota4-18`）を位置に関係なく抽出。末尾に付く記号（■◎等）は品番の一部として保持
- **高さ**: `高さ1.8m` `樹高H1.5m` `70cm` などを除去
- **捨てるリスト**: 送料無料・不在置き了承・程度 などの定型句を除去（画面で追加・編集可／端末ごとに保存）
- **※注記**: `※` 以降を除去
- 残りを **樹木名** とし、`【】`タグ・`「」`品種名・`大/中/小` は残す

## 出力

- **品番の英字** でグループ化して並べる（品番なしは末尾・空欄）
- 印刷される列は **済・樹木名・品番・メモ** の4つ（除外・確認は画面専用で紙に出ない）
- チェック欄（□）付き／印刷はインク節約（塗りつぶしなし・白黒）

## 画面の操作（除外・確認・メモ）

一覧表には、確認作業を助ける欄があります。

- **除外**（左端・画面のみ）: 掘り子に渡さない行にチェック。画面では薄く打消し線になり、**印刷には出ない**（行ごと消える）。件数表示に「除外 N 件 → 印刷 M 件」と反映
- **確認**（右端・画面のみ）: 目視確認できた行にチェックすると**薄緑にハイライト**。見落とし防止用（印刷には出ない）
- **メモ**（右側・印刷あり）: 手書き用の空欄。印刷物に出る
- 「除外」「確認」のチェックはその画面だけ（「変換する」でリセット／出荷日の変更では保持）
